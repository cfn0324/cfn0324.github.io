Markdown的基本语法

发表于 2024-02-12 更新于 2024-02-13

标题

用 # 号来表示 1-6 级标题

段落格式

字体

用 * 和 _ 来表示字体

分隔线

用三个以上的 * - _ 来表示分隔线

删除线

用 ~~ 来表示删除线

下滑线

用 < u > < /u > 来表示表示下划线

脚注

用[^]来表示脚注

列表

无序列表

用 * + - 来表示无序列表

有序列表

用 . 并加上数字来表示无有序列表

嵌套列表

在子列表中的选项前面加3个空格表示

区块

区块引用

用 > 来表示区块

区块嵌套

用 > 的数量来表示第几层嵌套

区块中使用列表

在列表前使用 >

列表中使用区块

在区块前使用列表

代码

片段代码

用 ` 来表示代码片段

代码区块

用 ``` 来表示代码区块并可以指定语言

链接

文本链接

用 [] 或者 <> 来表示需要被链接的文本，用 () 来提供链接

直接链接

用 <> 来直接使用链接

高级链接

我们可以通过变量来设置一个链接，变量赋值在文档末尾进行

[我的个人博客][1]

图片

在链接格式前加一个 ! 表示图片

ACM算法之博弈论

发表于 2024-02-12

巴什博弈

一堆n个物品，两个人轮流从中取出1~m个，最后取光者胜。

威佐夫博弈

两堆物品,两人轮流从一堆或者两堆中取相同1~不限个，最后取光者胜。

尼姆博弈

三堆物品，两人轮流取，每次取其中一堆1~不限个，最后取光者胜。

斐波那契博弈

一堆物n个，两人轮流取，不能取完，不能取超过上次两倍，最后取光者胜。
